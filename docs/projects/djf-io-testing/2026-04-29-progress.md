- `pnpm install`, `pnpm build`, `pnpm exec playwright install chromium`, `pnpm test:e2e` — all 7 tests pass (~3.6s)

## Decisions

- **Vitest deferred**: no shared/util code in djf.io worth unit-testing yet. Plan documents the trigger to add it later. Avoids fabricating helpers solely for tests.
- **Single browser (Chromium)**: blog is server-rendered HTML; cross-browser coverage adds CI minutes for marginal gain on this app. Easy to expand.
- **`retries: 2` in CI**: standard Playwright practice for flake mitigation; locally `0`.
