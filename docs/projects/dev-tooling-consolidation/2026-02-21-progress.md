# 2026-02-21 Progress

## Summary

Initial implementation of dev tooling consolidation: migrating shared dev tools to mise, upgrading Biome to 2.4 and Oxlint to 1.x, adding Prettier for Markdown/MDX formatting.

- Decided against oxfmt (no Astro support, non-JS formatting just delegates to Prettier anyway)
- Decided on: Biome for JS/TS/CSS/JSON, Prettier for MD/MDX only, both managed via mise
- Upgraded Biome config to 2.4 schema
- Updated Oxlint config for v1.x
- Added Prettier scoped to Markdown/MDX
- Consolidated .editorconfig to root
- Configured VS Code format-on-save
- Moved biome/oxlint from app devDeps to mise

## Decisions

- No pnpm workspace — apps keep independent lockfiles
- Dev tools installed via mise, not per-app devDependencies
- Prettier is only for .md and .mdx files; everything else uses Biome
- cspell stays in app devDeps for now (per-app dictionaries may diverge)

## Next steps

- Monitor Biome Markdown support progress (could eventually replace Prettier)
