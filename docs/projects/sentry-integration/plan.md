# Sentry Integration

## Goal

Wire Sentry into every app in the repo for crash and error monitoring.

## Scope

- All existing apps: djf.io, calendar-visualizer, ravrun
- All new domain sites (see [New Domain Sites](../new-domain-sites/plan.md))
- Per-app Sentry project, DSN, source maps, release tagging
- Confirm coverage works on Cloudflare Workers runtime
