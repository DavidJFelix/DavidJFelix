# 2026-06-17 — Research & planning session

## Summary

Researched [standard.site](https://standard.site) (an AT Protocol / Bluesky
interoperability standard for long-form publishing), audited `apps/djf.io`, and designed
the compatibility approach. Scope this session was **documentation only** — the plan and
the human-task issue — no app code written.

## Work completed

- Confirmed what standard.site is and what compatibility requires: a
  `site.standard.publication` record + one `site.standard.document` per post (on a PDS),
  plus website artifacts (`/.well-known/site.standard.publication` returning the
  publication AT-URI as plain text, and `<link rel="site.standard.document">` per post).
- Audited djf.io: Astro 6 static SSG on Cloudflare Workers; `post.id` (filename slug) is
  the universal join key; `BaseLayout.astro` already exposes `<slot name="head" />` and
  `BlogPost.astro` already injects JSON-LD through it; `rss.xml.ts` is the endpoint
  pattern to mirror; no `.well-known/` or `bin/` dir yet.
- Wrote [`plan.md`](./plan.md) with the full architecture and a 3-phase rollout.
- Filed a human-intervention GitHub issue for `@DavidJFelix` (DID + app password + GH secret).

## Decisions

- **Deterministic AT-URIs**: publication rkey `self`, document rkey = post slug. Every
  AT-URI is computable from one committed `ATPROTO_DID` constant + slug, so the build and
  the sync script agree without a PDS round-trip — no generated map, no bot commits. This
  is what makes CI-integrated sync clean.
- **Identity**: handle `@djf.io` on **bsky.social** (no self-hosted PDS). Records live in
  that account's repo under its `did:plc:…`.
- **Sync automation**: **CI-integrated on deploy** (David's choice over a local script).
  A bun `bin/sync-standard-site.ts` runs in `cd-deploy-djf-io.yml` after build, before
  deploy, gating deploy on sync success. Only secret is `ATPROTO_APP_PASSWORD`.
- **Endpoint**: Astro endpoint first (single source from the DID constant); static file
  in `public/.well-known/` as fallback if Astro excludes the leading-dot directory.
- **Out of scope**: `graph.subscription`, `graph.recommend`, `theme` (reader/client-side,
  not publisher-authored).
- **Deferred**: publication `icon` (needs a ≥256px square) and document `textContent`.
