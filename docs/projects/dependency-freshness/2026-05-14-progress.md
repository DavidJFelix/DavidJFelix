## Decisions made

- **Cadence**: weekly, all ecosystems.
- **Escalation**: one surface. A single `deps:needs-attention` label on PRs
  carrying a major or known-incompatible bump; a `## Skipped` section in the
  batch PR body for anything not applied. No companion issues for routine cases.
  True human-intervention blockers (credentials, replacement decisions) still
  follow the `CLAUDE.md` Human Intervention Task convention.
- **Lockfiles**: one per project. When both `pnpm-lock.yaml` and `bun.lock`
  exist, keep pnpm, delete `bun.lock`. `pnpm` is the accepted Node default;
  `bun` is preferred only where no Node toolchain is needed (Wrangler
  compatibility with bun-only is still an open question).
- **Tooling preferences** (now in `CLAUDE.md`): bun > pnpm > npm, yarn banned;
  `uv` for Python, pip/poetry banned; cargo; go mod; mise tasks > bun scripts >
  shell scripts, scripts >a few lines go in `bin/`, justfiles removed on sight,
  no new task tooling without an ask.
- **Deployment**: Vercel is dropped (moved to Cloudflare) — remove references
  when encountered.
- **No root workspace / monorepo orchestration.** If a workspace is ever wanted
  it lives in its own subdirectory.
