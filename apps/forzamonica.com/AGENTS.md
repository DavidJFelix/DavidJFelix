## Project Context

Headless Shopify storefront for forzamonica.com. TanStack Start on Cloudflare Workers; catalog and
cart come from the Shopify Storefront API (currently mock.shop), checkout is Shopify-hosted via the
cart's `checkoutUrl`.

## Key Technologies

- TanStack Start (React 19) with file-based routes in `src/routes/`
- PandaCSS — config recipes in `panda.config.ts`, codegen output in `styled-system/` (gitignored,
  regenerated by the `prepare` script)
- Ark UI for headless components, styled via Panda slot recipes (`QuantityField` is the reference
  pattern)
- Cloudflare Workers via `@cloudflare/vite-plugin`; bindings read with
  `import {env} from 'cloudflare:workers'` inside server-only code

## Architecture Notes

- All Storefront API calls go through server functions (`src/lib/shopify/`) so tokens stay
  server-side
- The cart ID is persisted in an httpOnly cookie; cart mutations re-run the route loader via
  `router.invalidate()`
- GraphQL documents and hand-written result types live in `src/lib/shopify/queries.ts`
